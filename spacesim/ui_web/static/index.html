          Help & reference

          Full UI reference (every control, dropdown, shortcut) lives in the training manual: docs/training/10-ui-reference.md.

          Keyboard shortcuts

          • Ctrl/Cmd-K — open the command palette
          • j / k — cycle through actor select
          • c — focus the compose form
          • g — graph the selected asset's telemetry
          • Esc — close the palette / coachmark

          Workflow

          • Click a fleet row to drill down. Shift-click to add to a multi-asset batch.
          • Compose an order, then ★ Save preset to keep it as a one-click playbook.
          • Pin a moment with 📌 Bookmark to jump back via the AAR scrubber.
          • ⏸ Handover blanks the screen for hot-seat handoffs between operators.
          • Inspect opens the YAML so trainers can fork the scenario.

          Acronyms

          • SOH — State of Health (bus/payload rollup)
          • RPO — Rendezvous & Proximity Operations
          • ISL — Inter-Satellite Link (crosslink)
          • SSN — Space Surveillance Network
          • SDA — Space Domain Awareness
          • EW — Electronic Warfare
          • ROE — Rules of Engagement
          • Δv — Delta-v (maneuver budget)
